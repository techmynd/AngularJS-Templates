WELCOME

This is an AngularJS app without jQuery and you are at homepage. I have covered following things in this app.

  1. Angular Setup along with Angular UI Bootstrap / Several Bootstrap components
  2. Routing / ng-view / custom 404 page
  3. Bootstrap Carousel
  4. ng-include
  5. ng-repeat with sort/search
  6. Show / Hide / Toggle
  7. To Do List
  8. Form Validation

Notes

I have used Angular UI Bootstrap (ui-bootstrap-tpls-1.3.2.min.js) and Bootstrap CSS from (Twitter Bootstrap v3.3.6).
I did not use JS that comes with Twitter Bootstrap v3.3.6
All the controllers are in assets/scripts/custom.js
This project is using routing so localhost or a server is required to run it.
Try a random non-existent route name in URL to see 404 page.
